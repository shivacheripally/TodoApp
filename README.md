# Todoapp

This repository contains the server-side implementation for the Todoapp project.

## Tech Stack

- EJS
- Express
- MongoDB / Mongoose
- CSS
- JavaScript
- Backend

## Code-Based Notes

- MongoDB persistence is configured through Mongoose models.

- `/.gitignore`

- `/package-lock.json`

## Local Setup

1. Install dependencies with `npm install`.
2. Start the project with `npm start`.

## Available Scripts

- `npm run start` or `npm start` as defined in `package.json`
- `npm run test` or `npm test` as defined in `package.json`
